# 🚀 Worklyn

Worklyn is a multi-tenant SaaS workspace platform designed for team collaboration, project management, and task tracking.

It enables teams to create workspaces, manage projects, assign tasks, and collaborate efficiently with role-based access control.

---

## ✨ Features

- 🏢 Multi-tenant workspace system  
- 👥 Role-based access control (Admin / Member)  
- 🔐 Authentication (Google & GitHub OAuth)  
- 📁 Project management system  
- ✅ Task management with status tracking (Todo, In Progress, Done)  
- 📊 Workspace dashboard overview  
- 🧠 Activity tracking system (planned)  
- ⚡ Clean and responsive UI

## 🧱 Tech Stack

- Next.js (App Router)
- TypeScript
- PostgreSQL
- Prisma ORM
- NextAuth.js (Google & GitHub OAuth)
- Shadcn UI
- Lucide React
- Zod (validation)
- TanStack Query

---

## 🏗️ Architecture

Worklyn is built using a **multi-tenant architecture**, where all data is scoped to a workspace.

- Users belong to workspaces  
- Projects belong to workspaces  
- Tasks belong to projects  
- Strict data isolation ensures secure separation between organizations
